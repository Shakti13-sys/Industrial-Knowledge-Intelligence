## Overview

The Industrial-Knowledge-Intelligence Project (IKIP) is an intelligent system designed to streamline the management, processing, and querying of industrial knowledge embedded within documents like PDFs. Leveraging the power of Artificial Intelligence, specifically Large Language Models (LLMs) via OpenAI and LangChain, IKIP empowers users to extract critical insights and obtain precise answers from their vast repositories of structured and unstructured industrial documentation.

This project features a robust backend API built with FastAPI for handling document processing and AI interactions, complemented by a modern, responsive frontend developed with Next.js and React for an intuitive user experience.

## Features

*   **PDF Document Processing**: Effortlessly upload PDF documents, and the system automatically extracts textual content for analysis.
*   **Knowledge Base Management**: Stores and manages extracted document content and allows for the definition of custom entities, forming a rich industrial knowledge base.
*   **Natural Language Querying**: Interact with your industrial knowledge base using natural language questions to retrieve specific information.
*   **AI-Powered Insights**: Get intelligent responses and detailed insights generated by OpenAI models, orchestrated through the LangChain framework.
*   **Modern User Interface**: A user-friendly, responsive, and interactive web interface built with Next.js and React, styled with Tailwind CSS.
*   **CORS Enabled API**: The backend API is configured with CORS, allowing flexible deployment and seamless communication with the frontend.
*   **Secure Configuration**: Utilizes environment variables (`.env`) for managing sensitive configurations like API keys, ensuring security.

## Tech Stack

IKIP is built using a modern full-stack architecture:

**Backend:**
*   **Python**: Primary programming language.
*   **FastAPI**: High-performance web framework for building APIs.
*   **Pydantic**: Data validation and settings management.
*   **PyPDF2**: For PDF document parsing and text extraction.
*   **LangChain**: Framework for developing applications powered by language models.
*   **OpenAI API**: Integration with OpenAI's powerful language models.
*   **python-dotenv**: For managing environment variables.

**Frontend:**
*   **TypeScript / JavaScript**: Programming languages for the web interface.
*   **Next.js**: React framework for production-grade web applications.
*   **React**: JavaScript library for building user interfaces.
*   **Tailwind CSS**: Utility-first CSS framework for rapid UI development.
*   **Axios**: Promise-based HTTP client for the browser and Node.js.

## Installation

Follow these steps to set up and run the Industrial-Knowledge-Intelligence project locally.

### Prerequisites

*   Python 3.8+
*   Node.js (LTS recommended) & npm (or yarn)
*   An OpenAI API Key

### 1. Clone the Repository

```bash
git clone https://github.com/Shakti13-sys/Industrial-Knowledge-Intelligence.git
cd Industrial-Knowledge-Intelligence
```

### 2. Backend Setup

Navigate to the `backend` directory and set up the Python environment.

```bash
cd backend
```

**Create a Virtual Environment:**

```bash
python -m venv venv
```

**Activate the Virtual Environment:**

*   **macOS/Linux:**
    ```bash
    source venv/bin/activate
    ```
*   **Windows:**
    ```bash
    venv\Scripts\activate
    ```

**Install Dependencies:**

Create a `requirements.txt` file in the `backend` directory with the following content:

```
fastapi==0.111.0
uvicorn==0.30.1
pydantic==2.7.4
python-dotenv==1.0.1
PyPDF2==3.0.1
langchain-openai==0.1.13
```

Then install them:

```bash
pip install -r requirements.txt
```

**Configure Environment Variables:**

Create a `.env` file in the `backend` directory and add your OpenAI API key:

```
OPENAI_API_KEY="YOUR_OPENAI_API_KEY"
```
Replace `"YOUR_OPENAI_API_KEY"` with your actual key.

### 3. Frontend Setup

Open a new terminal, navigate to the `frontend` directory, and install Node.js dependencies.

```bash
cd ../frontend # If you are still in the backend directory
# or: cd Industrial-Knowledge-Intelligence/frontend
```

**Install Dependencies:**

```bash
npm install
# or yarn install
```

## Usage

### 1. Start the Backend API

Make sure your Python virtual environment for the backend is active.

```bash
cd backend
# Activate virtual environment if not already active
# source venv/bin/activate (macOS/Linux)
# venv\Scripts\activate (Windows)

uvicorn main:app --reload
```
The backend API will be running at `http://127.0.0.1:8000`.

### 2. Start the Frontend Application

Open a **new terminal**, navigate to the `frontend` directory.

```bash
cd frontend

npm run dev
# or yarn dev
```
The frontend application will be available at `http://localhost:3000`.

Open your web browser and visit `http://localhost:3000` to access the Industrial Knowledge Intelligence platform. You can now upload PDF documents, manage entities, and interact with the AI query engine.
